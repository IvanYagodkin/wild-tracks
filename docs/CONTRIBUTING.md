## Как добавить фото следа

1. Сфотографируйте след сверху, рядом положите масштабный объект (линейку, монету).
2. Сделайте ещё один снимок — с общим планом местности.
3. Загрузите фото в папку `data/images/вид_животного/` (например, `fox` или `wolf`).
4. Добавьте запись в файл `data/metadata/observations.csv`.
